Steps

  • For each node N:
    • initialize N’s visited mark to false
    • initialize N’s path to null
  • create minimum priority queue pq for paths
  • pq.insert( [ S->S: 0 ] )
  • while ( !pq.isEmpty() ):
    • [ S->...->C: Cost ] = pq.removeMin()
    • if C’S path == null:
      • set C’s visited mark to true
      • set C’s shortest path to [ S->...->C: Cost ]
      • for each edge with weight W to unvisited successor U of C:
        • pq.insert( [S->...->C->U: Cost + W] )
